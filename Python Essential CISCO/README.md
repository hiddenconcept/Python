# 🐍 Cisco Python Essentials 1

## Overview

This repository contains my coursework, exercises, and programming projects completed while studying **Python Essentials 1** through the Cisco Networking Academy.

The course focused on building a solid foundation in Python programming, beginning with basic syntax and progressing through functions, modules, file handling, exceptions, and problem-solving using real programming concepts.
